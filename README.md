# Sentiment_analysis

# 📊 Trader Performance vs Market Sentiment Analysis

---

## 🚀 Project Overview

This project analyzes the relationship between cryptocurrency trader performance and market sentiment (**Fear, Greed, Neutral, Extreme Fear, Extreme Greed**).

The goal is to understand how market emotions influence trading behavior, profit/loss, risk-taking, and success rate using real trading data.

---

## 🎯 Objective

- Explore relationship between market sentiment and trader performance  
- Identify hidden patterns in trading behavior  
- Analyze profit/loss across sentiment conditions  
- Study risk behavior using leverage and position size  
- Determine if sentiment can predict trading success  

---

## 📁 Datasets Used

### 1. Trader Dataset
- Account  
- Coin  
- Execution Price  
- Size Tokens / Size USD  
- Side (Buy/Sell)  
- Start Position  
- Direction (Long/Short)  
- Closed PnL  
- Fee  
- Timestamp  

### 2. Market Sentiment Dataset
- Date  
- Fear & Greed Index Value  
- Classification:
  - Fear  
  - Greed  
  - Neutral  
  - Extreme Fear  
  - Extreme Greed

## 🔥 Key Insights

- Market sentiment strongly impacts trader behavior  
- Greed periods lead to higher leverage usage (high risk)  
- Fear periods show more cautious trading behavior  
- Buy trades perform better in bullish sentiment  
- Sell trades perform better in bearish sentiment  
- A small number of traders contribute most of the profit  
- Trading fees significantly affect net profitability  
- Sentiment can partially predict winning probability

## 📉 Conclusion

Market sentiment is a strong behavioral indicator that influences:

- Risk-taking behavior

This analysis proves that emotional market conditions directly affect trading outcomes, making sentiment a valuable feature for trading strategies.
